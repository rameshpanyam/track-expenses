Implementation notes

  • Single source of truth — all colors, radii, shadows, and motion live as CSS custom properties on :root. Switching themes is one class toggle (.dark).
  • Numerals — all money displays use font-variant-numeric: tabular-nums and JetBrains Mono ligatures for sub-pixel-stable columns.
  • Iconography — Phosphor-style 1.6px strokes inline as SVG; replaces every emoji in app chrome (kept emoji only inside category labels per user research).
  • Sparklines — pure SVG paths driven by CSS variables; no chart library needed, <1KB each.
  • Hero card — charcoal surface with dual radial-gradient (emerald + indigo) gives depth without skeuomorphism.
  • Dark mode — every token has a paired value in .dark; verified contrast ≥ 4.5:1 for body and ≥ 3:1 for large text.
  • Modal — bottom-sheet pattern with grab-handle, dimmed backdrop, sticky CTA bar. Replaces the current full-screen overlay.
  • Empty state — every section has a hand-designed empty state; "Upload PDF" exposed as a first-class action to surface the killer feature.
